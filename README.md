# Dashboard de Crimes Raciais no Distrito Federal

Projeto acadêmico de Business Intelligence e Data Warehouse desenvolvido utilizando Power BI para análise de crimes de racismo e injúria racial no Distrito Federal.

## Objetivo

Analisar dados públicos por meio de dashboards interativos e análises temporais, territoriais e comparativas, auxiliando na visualização estratégica da criminalidade racial no DF.

## Tecnologias Utilizadas

* Power BI
* Power Query
* DAX
* Modelagem Dimensional (Star Schema)

## Funcionalidades

* Evolução temporal dos crimes
* Comparativo entre racismo e injúria racial
* Mapa geográfico por Região Administrativa
* Meta analítica de redução criminal
* Classificação de intensidade criminal
